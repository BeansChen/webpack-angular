# webpack-angular
此前用angularJS很长一点时间了，但一直没有使用过什么构建工具或者是模块化打包工具，
随着es6的到来，webpack的火热，因此有了改变架构方案的念头，此项目为angularJS的模块化方案，
脚手架根据vue-cli改写。


## 下载

下载安装包解压或者 git clone https://github.com/BeansChen/webpack-angular.git

## 安装依赖

    npm install
    
## 生产模式运行

    npm run dev
    
## 打包

    npm run build
