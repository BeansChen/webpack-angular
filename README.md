# webpack-angular
> 前阵子本想自己做个博客系统，需要在网上找个webpack打包angular的例子，结果没有找到，于是参考vue-cli做了一套。然而，博客最终用了github pages来搭建，所以就没有用上，这里我把我做的分享出来...,只需要几步就可以看到效果了

## 下载

下载安装包解压或者 git clone https://github.com/BeansChen/webpack-angular.git

## 安装依赖

    npm install
    
## 生产模式运行

    npm run dev
    
## 打包

    npm run build
